# Mobile App (Wave 4)

This directory will contain the Flutter/React Native mobile app.

## Getting Started (Future)

### Flutter

```bash
cd mobile-app
flutter pub get
flutter run
```

```
mobile-app/

│   ├── services/        # API, Auth, Notifications
│   └── models/          # Data models
└── pubspec.yaml
```
