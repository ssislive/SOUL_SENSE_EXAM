# LLM Prompt Templates (Wave 7)

This directory contains prompt templates for LLM integrations.

## Files

| File                 | Purpose                                     |
| -------------------- | ------------------------------------------- |
| `mental_health.txt`  | Prompts for emotional support conversations |
| `task_planner.txt`   | Prompts for task breakdown                  |
| `virtual_friend.txt` | Prompts for the Virtual Friend agent        |
| `system.txt`         | Base system prompt with safety constraints  |
