# Web Frontend (Wave 4)

This directory will contain the Next.js web client.

## Getting Started (Future)

```bash
cd frontend-web
npm install
npm run dev
```

## Structure

```
frontend-web/
├── src/
│   ├── pages/           # Page routes
│   ├── components/      # Reusable components
│   └── services/        # API client
├── styles/
└── package.json
```

See `FILE_ARCHITECTURE.md` for detailed file mappings.
